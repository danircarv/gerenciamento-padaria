PROJECT OVERVIEW
----------------
A complete web-based bakery management system built with Spring Boot 3.2, 
JDBC (without JPA), and MySQL 8. Implements a 3-layer architecture with 
comprehensive business logic, REST API, and web interface.

TECHNOLOGY STACK
----------------
- Java 17
- Spring Boot 3.2.0
- Spring JDBC (JdbcTemplate - NO JPA)
- MySQL 8
- Thymeleaf Template Engine
- Bootstrap 5
- Maven
- Docker & Docker Compose
- JUnit 5 + Mockito

BUSINESS FLOWS
--------------
Sales Flow:
1. Select products and quantities
2. System checks stock availability
3. Calculate totals (with discount support)
4. Register sale
5. Automatically update stock
6. Allow cancellation (restores stock)

Order Flow:
1. Customer places order with delivery date
2. System registers order (status: PENDENTE)
3. Status progression: PENDENTE → CONFIRMADA → EM_PRODUCAO → PRONTA → ENTREGUE
4. Support for down payment
5. Order details in observations

Stock Control Flow:
1. System monitors stock levels
2. Automatic alerts when quantity < minimum
3. Manual stock entries
4. Automatic updates on sales
5. Stock shortage reports
